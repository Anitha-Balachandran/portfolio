Recipe Genie

Description

  • Anitha Balachandran

  • Code

  • Sep, 2024

This project develops an intelligent recipe recommendation system that suggests dishes based on user-input ingredients. Using TF-IDF and natural language processing, it matches ingredients with a database of recipes, enabling users to explore diverse meal options easily.
